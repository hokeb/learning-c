# Exercise 3.01

What output do the following calls to printf produce?

```text
(a) printf("%6d, %4d", 86, 1040);
(b) printf("%12.5e", 30.253);
(c) printf("%.4f", 83.162);
(d) printf("%-6.2g", .0000009979);
```

## Solution

&#176; - denotes a space

(a) &#176;&#176;&#176;&#176;86, 1040

(b) &#176;3.02530+01

(c) 83.1620

(d) 1e-06&#176;
